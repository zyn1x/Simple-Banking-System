A simple Banking system that is built using the concept of OOPs in python.
A user can open 3 types of bank account- a simple bank account, a interest rate bank account, a saving account
a user can do the following transactions:
  1) get balance of account
  2) deposit money 
  3) withdraw money
  4) transfer money
in case of interest rate account, the user will get an additional amount of interest rate along with the deposited amount
in case of interest rate account, the user will have to pay a certain fee everytime he withdraws money
